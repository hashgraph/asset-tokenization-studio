## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `Breadcrumb.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    Breadcrumb: ConfigBreadcrumb,
  },
};
```

```tsx dark
const ConfigBreadcrumb: BreadcrumbThemeConfiguration = {
  parts: breadcrumbPartsList,
  baseStyle: {},
};
```

### Component parts

You have these parts available: `"container" | "isDesktop" | "item" | "menu" | "link" | "separator"` (`breadcrumbPartsList`)
