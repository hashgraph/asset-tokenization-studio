## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `Header.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    Header: ConfigHeader,
  },
};
```

```tsx dark
const ConfigHeader: HeaderThemeConfiguration = {
  parts: headerPartsList,
  baseStyle: {},
};
```

### Component parts

You have these parts available: `"container" | "contentContainer"` (`headerPartsList`)
