## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `PageTitle.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    PageTitle: ConfigPageTitle,
  },
};
```

```tsx dark
const ConfigPageTitle: PageTitleThemeConfiguration = {
  parts: pageTitlePartsList,
  baseStyle: {},
};
```

### Component parts

You have these parts available: `"backButton" | "textTitle" | "container"` (`pageTitlePartsList`)
