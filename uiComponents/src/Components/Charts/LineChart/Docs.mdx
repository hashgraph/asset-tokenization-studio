## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `LineChart.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    LineChart: ConfigLineChart,
  },
};
```

```tsx dark
const ConfigLineChart = {
  parts: lineChartPartsList,
  baseStyle: {},
};
```

### Component parts

You have these parts available: `"grid" | "xAxisLine" | "xAxisLabel" | "yAxisLine" | "yAxisLabel" | "series"` (`lineChartPartsList`)
