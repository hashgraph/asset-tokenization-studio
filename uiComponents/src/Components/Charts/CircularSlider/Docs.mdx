## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `CircularSlider.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    CircularSlider: ConfigCircularSlider,
  },
};
```

```tsx dark
const ConfigCircularSlider: CircularSliderThemeConfiguration = {
  parts: circularSliderPartsList,
  baseStyle: {},
};
```

### Component parts

You have these parts available: `"label" | "track"` (`circularSliderPartsList`)
