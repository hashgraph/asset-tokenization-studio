## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `BarChart.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    BarChart: ConfigBarChart,
  },
};
```

```tsx dark
const ConfigBarChart: BarChartThemeConfiguration = {
  parts: barChartPartsList,
  baseStyle: {},
};
```

### Component parts

You have these parts available: `"label" | "track" | "filledTrack"` (`barChartPartsList`)
