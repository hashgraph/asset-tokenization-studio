## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `IconButton.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    IconButton: ConfigIconButton,
  },
};
```

```tsx dark
const ConfigIconButton: IconButtonThemeConfiguration = {
  baseStyle: {},
  sizes: {},
  variants: {},
  defaultProps: {},
};
```
