## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `ClipboardButton.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    ClipboardButton: ClipboardButtonTheme,
  },
};
```

```tsx dark
const ClipboardButtonTheme: ClipboardButtonConfiguration = {
  parts: clipboardButtonPartsList,
  baseStyle: {},
};
```

### Component parts

You have these parts available: `"iconButton" | "icon" | "tooltip"` (`clipboardButtonPartsList`)
