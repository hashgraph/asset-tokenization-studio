## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `CardButton.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    CardButton: ConfigCardButton,
  },
};
```

```tsx dark
const ConfigCardButton: CardButtonThemeConfiguration = {
  parts: cardButtonPartsList
  baseStyle: {},
};
```

### Component parts

You have these parts available: `"container" | "content" | "icon" | "text" | "check" | "tooltip"` (`cardButtonPartsList`)
