## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `PopUp.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    PopUp: ConfigPopUp,
  },
};
```

```tsx dark
const ConfigPopUp: PopUpThemeConfiguration = {
  parts: popUpPartsList,
  baseStyle: {},
};
```

### Component parts

You have these parts available: `"overlay" | "container"  | "dialog"  | "header" | "closeButton"  | "body"  | "footer"  | "icon"  | "title"  | "description"  | "contentContainer"  | "actionButton"` (`popUpPartsList`)

## How to use

### Default

```tsx
<PopUp isOpen isOpen title='title' description='description' />
```

### Custom organization

```tsx
<PopUp isOpen>
  <PopupText type='description' label='title' />
  <PopupText type='title' label='description' />
  <PoupIcon />
</PopUp>
```

### Confirmation Popup

```tsx
<ConfirmationPopUp
  isOpen={isOpen}
  onClose={onClose}
  onCancel={handleCancel}
  onConfirm={handleConfirm}
  confirmationText='CONFIRMATION TEXT'
  title='Title'
  description='Description'
  label="Type 'CONFIRMATION TEXT' to confirm"
/>
```
