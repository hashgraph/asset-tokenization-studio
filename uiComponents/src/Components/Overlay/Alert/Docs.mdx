## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `Alert.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    Alert: ConfigAlert,
  },
};
```

```tsx dark
const ConfigAlert: AlertThemeConfiguration = {
  parts: alertPartsList,
  baseStyle: {},
};
```

### Component parts

You have these parts available: `"container" | "contentContainer" | "title" | "description" | "icon" | "spinner" | "closeBtn` (`alertPartsList`)
