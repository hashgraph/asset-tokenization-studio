## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `FileInput.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    FileInput: ConfigFileInput,
  },
};
```

```tsx dark
const ConfigFileInput = {
  parts: fileInputPartsList,
  baseStyle: {},
};
```

### Component parts

You have these parts available: `"container" | "icon" | "label" | "description" | "input"` (`fileInputPartsList`)
