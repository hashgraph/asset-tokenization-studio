## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `Toggle.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    Switch: ConfigToggle,
  },
};
```

```tsx dark
const ConfigToggle: ToggleThemeConfiguration = {
  parts: togglePartsList,
  baseStyle: {},
};
```

### Component parts

You have these parts available: `"container" | "track" | "thumb" | "label"` (`togglePartsList`)
