## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `Textarea.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    Textarea: ConfigTextarea,
  },
};
```

```tsx dark
const ConfigTextarea: TextareaThemeConfiguration = {
  parts: textareaPartsList,
  baseStyle: {},
};
```

### Component parts

You have these parts available: `"labelContainer" | "label" | "container" | "length"` (`textareaPartsList`)
