## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `Radio.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    Radio: ConfigRadio,
  },
};
```

```tsx dark
const ConfigRadio: RadioThemeConfiguration = {
  parts: radioPartsList,
  baseStyle: {},
};
```

### Component parts

You have these parts available: `"control" | "label" | "container"` (`radioPartsList`)
