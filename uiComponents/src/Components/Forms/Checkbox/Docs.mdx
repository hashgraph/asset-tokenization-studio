## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `Checkbox.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    Checkbox: ConfigCheckbox,
  },
};
```

```tsx dark
const ConfigCheckbox: CheckboxThemeConfiguration = {
  parts: checkboxPartsList,
  baseStyle: {},
};
```

### Component parts

You have these parts available: `"icon" | "label" | "container" | "control" | "iconCustom"` (`checkboxPartsList`)
