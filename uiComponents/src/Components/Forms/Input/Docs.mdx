## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `Input.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    Input: ConfigInput,
  },
};
```

```tsx dark
const ConfigInput: InputThemeConfiguration = {
  parts: inputPartsList,
  baseStyle: {},
};
```

### Component parts

You have these parts available: `"labelContainer" | "label" | "addonLeft" | "addonRight" | "input" | "container" | "errorIcon"` (`inputPartsList`)
