## How to set up

Select is made using the styles of the following components: [Input](https://www.figma.com/file/YF8AvIq0MclfhGDvKgzypp/ioBricks-Design-System?type=design&node-id=1503%3A22828&mode=design&t=KeI2zscsNFcspEfb-1) & [Dropdown](https://www.figma.com/file/YF8AvIq0MclfhGDvKgzypp/ioBricks-Design-System?type=design&node-id=3310%3A38931&mode=design&t=KlerdyEMWSQWLAVX-1). Add the following configurations to the `theme.tsx`, `Input.ts` and `Dropdown.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    Input: ConfigInput,
    Dropdown: ConfigDropdown,
  },
};
```
