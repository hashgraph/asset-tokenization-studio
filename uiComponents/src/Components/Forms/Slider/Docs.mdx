## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `Slider.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    Slider: ConfigSlider,
  },
};
```

```tsx dark
const ConfigSlider: SliderThemeConfiguration = {
  parts: sliderPartsList,
  baseStyle: {},
};
```

### Component parts

You have these parts available: `"filledTrack" | "track" | "container" | "thumb"` (`sliderPartsList`)
