## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `Stepper.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    Stepper: ConfigStepper,
  },
};
```

```tsx dark
const ConfigStepper: StepperThemeConfiguration = {
  parts: stepperPartsList,
  baseStyle: {},
};
```

### Component parts

You have these parts available: `"description" | "icon" | "indicator"|  "number" | "separator" | "title" | "step" | "stepper" | "stepTextContainer" | "stepContainer"` (`stepperPartsList`)
