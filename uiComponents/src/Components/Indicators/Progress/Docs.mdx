## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `Progress.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    Progress: ConfigProgress,
  },
};
```

```tsx dark
const ConfigProgress: ProgressThemeConfiguration = {
  parts: progressPartsList,
  baseStyle: {},
  defaultProps: {},
};
```

### Component parts

You have these parts available: `"label" | "track" | "filledTrack"` (`progressPartsList`)
