## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `PanelTitle.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    PanelTitle: ConfigPanelTitle,
  },
};
```

```tsx dark
const ConfigPanelTitle: PanelTitleThemeConfiguration = {
  parts: panelTitlePartsList,
  baseStyle: {},
};
```

### Component parts

You have these parts available: `"title" | "container"` (`panelTitlePartsList`)
