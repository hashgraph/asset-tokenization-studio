## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `InfoDivider.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    InfoDivider: ConfigInfoDivider,
  },
};
```

```tsx dark
const ConfigInfoDivider: InfoDividerThemeConfiguration = {
  parts: infoDividerPartsList,
  baseStyle: {},
  sizes: {},
};
```

### Component parts

You have these parts available: `"container" | "titleContainer" | "number" | "step" | "title" | "divider"` (`infoDividerPartsList`)
