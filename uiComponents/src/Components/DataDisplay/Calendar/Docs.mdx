## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `Calendar.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    Calendar: ConfigCalendar,
  },
};
```

```tsx dark
const ConfigCalendar: CalendarThemeConfiguration = {
  parts: calendarPartsList,
  baseStyle: {},
};
```

### Component parts

You have these parts available: `"footer" | "header" | "container" | "headerTitle" | "day" | "month" | "yearTitle" | "dropdownButton" | "dropdownPanel" | "changeMonthButton" | "timeInput" | "input" | "todayTooltip"` (`calendarPartsList`)

## How to use

```tsx
import { Calendar } from "@hashgraph/uiComponents/DataDisplay/Calendar";
```

```tsx
<Calendar selected={selectedDate} onSelect={setSelectedDate} />
```

### Disabled weekends

```tsx
<Calendar selected={selectedDate} onSelect={setSelectedDate} disabledWeekends />
```

### Disabled week days

Disable weekdays (0: Sunday,1: Monday,2: Tuesday,3: Wednesday,4: Thursday,5: Friday,6: Saturday)

```tsx
<Calendar
  selected={selectedDate}
  onSelect={setSelectedDate}
  disabledWeekdays={[0, 6]}
/>
```

### Disabled dates

```tsx
<Calendar
  selected={selectedDate}
  onSelect={setSelectedDate}
  disabledDates={[new Date(2023, 5, 12), new Date(2023, 5, 15)]}
/>
```
