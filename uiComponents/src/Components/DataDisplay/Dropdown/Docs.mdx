## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `Dropdown.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    Dropdown: ConfigDropdown,
  },
};
```

```tsx dark
const ConfigDropdown: DropdownThemeConfiguration = {
  parts: dropdownPartsList,
  baseStyle: {},
};
```

### Component parts

You have these parts available: `"wrapper" | "container" | "itemIconWeight" | "itemLabel" | "itemContainer"` (`dropdownPartsList`)

## How to use

### Dropdown

Make sure to wrap the `Dropdown` component inside a `Menu` component from `@chakra-ui`

The trigger to open it should be wrapped in a `MenuButton` from `@chakra-ui`

```tsx dark
<Menu>
  <MenuButton>
    <Avatar />
  </MenuButton>
  <Dropdown>
    <DropdownItem label="Item" isActive />
    <DropdownItem label="Item" />
    <DropdownItem label="Item" />
    <DropdownItem label="Item" />
  </Dropdown>
</Menu>
```

### DropdownItem

Use this component inside the `Dropdown` component.

```tsx dark
<DropdownItem icon={Cookie} label="Item" />
```

### I only want the style of the dropdown

If you only want the look and feel of the component, but you want to implement it in other component because the `Dropdown` doesn't fit your needs, you can use:

```tsx dark
const styles = useChakraMultiStyleConfig("Dropdown", {
  variant,
}) as DropdownThemeStyle;
```
