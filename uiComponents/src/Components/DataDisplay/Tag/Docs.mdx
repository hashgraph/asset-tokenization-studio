## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `Tag.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    Tag: ConfigTag,
  },
};
```

```tsx dark
const ConfigTag: TagThemeConfiguration = {
  parts: tagPartsList,
  baseStyle: {},
};
```

### Component parts

You have these parts available: `"container" | "label"` (`tagPartsList`)
