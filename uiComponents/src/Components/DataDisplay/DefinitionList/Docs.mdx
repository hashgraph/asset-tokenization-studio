## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `DefinitionList.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    DefinitionList: ConfigDefinitionList,
  },
};
```

```tsx dark
const ConfigDefinitionList: DefinitionListThemeConfiguration = {
  parts: definitionListPartsList,
  baseStyle: {},
};
```

### Component parts

You have these parts available: `"definitionListGrid" | "container" | "listTitle" | "listItem" | "listItemTitle" | "listItemDescription"` (`definitionListPartsList`)
