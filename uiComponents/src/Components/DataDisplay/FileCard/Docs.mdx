## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `FileCard.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    FileCard: ConfigFileCard,
  },
};
```

```tsx dark
const ConfigFileCard: FileCardThemeConfiguration = {
  parts: dropdownPartsList,
  baseStyle: {},
};
```

### Component parts

You have these parts available: `"container" | "name" | "size" | "closeIcon"` (`fileCardPartsList`)
