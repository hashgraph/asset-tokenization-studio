## How to set up

### Config theme

Add the following configurations to the `theme.tsx`, `Accordion.ts` and `AccordionItem.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    Accordion: ConfigAccordion,
    AccordionItem: ConfigAccordionItem,
  },
};
```

```tsx dark
const ConfigAccordion: AccordionThemeConfiguration = {
  parts: accordionPartsList,
  baseStyle: {},
};
```

```tsx dark
const ConfigAccordionItem: AccordionItemThemeConfiguration = {
  parts: accordionItemPartsList,
  baseStyle: {},
};
```

### Component parts

You have these parts available:

- `"container", "title", "item"` (`accordionPartsList`)
- `"button", "item", "panel"` (`accordionItemPartsList`)
