## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `Tabs.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    Tabs: ConfigTabs,
  },
};
```

```tsx dark
const ConfigTabs: TabsThemeConfiguration = {
  parts: tabsPartsList,
  baseStyle: {},
};
```

### Component parts

You have these parts available: `"root" | "tab" | "tablist" | "tabpanel" | "tabpanels" | "indicator"` (`tabsPartsList`)
