## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `DetailReview.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    DetailReview: ConfigDetailReview,
  },
};
```

```tsx dark
const ConfigDetailReview: DetailReviewThemeConfiguration = {
  parts: detailReviewPartsList,
  baseStyle: {},
};
```

### Component parts

You have these parts available: `"title" | "value" | "container"` (`detailReviewPartsList`)
