## How to set up

Add the following configurations to the `theme.tsx` and `Logo.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    Logo: ConfigLogo,
  },
};
```

```tsx dark
const ConfigLogo = {
  baseStyle: {
    isoImage: logoIso,
    fullImage: logoFull,
  },
};
```
