## How to set up

### Config theme

Add the following configurations to the `theme.tsx` and `Avatar.ts` files, respectively.

```tsx dark
const theme = {
  components: {
    Avatar: ConfigAvatar,
  },
};
```

```tsx dark
const ConfigAvatar: AvatarThemeConfiguration = {
  parts: avatarPartsList,
  baseStyle: {},
  sizes: {},
  variants: {},
};
```

### Component parts

You have these parts available: `"container" | "badge" | "label" | "group" | "excessLabel"` (`avatarPartsList`)
