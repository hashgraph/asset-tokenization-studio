# Asset Tokenization Studio (ATS)

Create and manage tokenized securities on the Hedera network with built-in compliance and regulatory features.

## What is ATS?

Asset Tokenization Studio is a complete platform for issuing, managing, and trading digital securities. It enables you to tokenize real-world assets like company shares and bonds while maintaining full regulatory compliance.

### Key Benefits

| Benefit                 | Description                                                                     |
| ----------------------- | ------------------------------------------------------------------------------- |
| **Launch Faster**       | Ready-to-use infrastructure for security tokens - no need to build from scratch |
| **Stay Compliant**      | Built-in KYC/AML, transfer restrictions, and audit trails                       |
| **Automate Operations** | Execute corporate actions like dividends and coupons automatically              |
| **Scale Confidently**   | Enterprise-grade architecture with institutional custody support                |
| **Maintain Control**    | Granular permissions, freeze controls, and emergency stops                      |

## Platform Components

ATS provides everything you need to tokenize securities:

| Component           | Description                                               | For Who        |
| ------------------- | --------------------------------------------------------- | -------------- |
| **Web Application** | User-friendly interface for token creation and management | Business users |
| **TypeScript SDK**  | Programmatic access to all platform features              | Developers     |
| **Smart Contracts** | Modular, upgradeable contracts on Hedera                  | Developers     |

## Standards & Compliance

ATS implements industry-standard protocols for security tokens:

- **ERC-1400**: Security Token Standard for regulated securities
- **ERC-3643 (T-REX)**: Token for Regulated EXchanges with on-chain compliance
