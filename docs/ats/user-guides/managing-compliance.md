# Managing Compliance and KYC

Learn how to configure and manage compliance rules, KYC verification, and transfer restrictions.

## Overview

ATS provides comprehensive compliance features:

- KYC/AML verification integration
- Transfer restrictions by country or investor type
- Whitelist and blacklist management
- Compliance rule enforcement

## KYC Management

### Adding KYC Providers

Configure identity verification providers:

- Select KYC provider from supported list
- Configure API credentials
- Set verification requirements

### Verifying Investors

Process to verify token holders:

1. Investor submits KYC information
2. Provider verifies identity
3. Approved investors added to whitelist
4. Tokens can be transferred to verified addresses

## Transfer Restrictions

### Country Restrictions

Configure jurisdiction rules:

- Whitelist allowed countries
- Blacklist restricted jurisdictions
- Set default transfer policies

### Investor Type Restrictions

Limit transfers based on investor classification:

- Accredited investors only
- Institutional investors
- Retail investors (if allowed)

## Managing Whitelists

Add or remove approved addresses:

- View current whitelist
- Add new addresses
- Remove addresses
- Bulk import/export

## Compliance Reporting

Generate compliance reports:

- Holder registry with KYC status
- Transfer history and restrictions applied
- Failed transfer attempts
- Audit logs
